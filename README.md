# 🎮 Procrastination Simulator


A chaotic, interactive web app that simulates what *actually* happens when you try to be productive.

---

## 🚀 Features

### 🧠 Fake Productivity Mode

* Start a study session 
* Clean, minimal UI 

### 📉 Distraction System

* Blurry memes appear over time
* Clicking them reveals the meme **but increases your distraction score**


### 😂 Meme Integration

* Fetches random memes from public APIs
* Infinite, unpredictable distraction fuel

### 🎮 Mini Games

* 🦖 Dino Game (Chrome-style)
* 🐦 Flappy Bird clone

### ⏱️ Timer Illusion

* A study timer that *technically* runs…

---

## 🧩 Tech Stack

**Frontend**

* React

**Backend**

* Django
* Django REST Framework

**APIs**

* Public Meme APIs

---

## 🖥️ How It Works

1. User starts a “study session”
2. Timer begins
3. Distractions start appearing:

   * Blurry memes
   * Mini-games
4. User interacts → distraction score increases
5. Reality check: you didn’t study

---

## 📊 Core Concept

This project is a playful take on:

* procrastination
* short attention spans
* digital distractions

It flips the idea of productivity apps by doing the exact opposite.

---

## 🛠️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/9lixir/Procrastination-game.git
cd Procrastination-game
```

### 2. Backend Setup (Django)

```bash
cd backend
pip install -r requirements.txt
python manage.py runserver
```

### 3. Frontend Setup (React)

```bash
cd frontend
npm install
npm start
```
